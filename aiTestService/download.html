테스트 시작하기

테스트를 시작하기 전에

mTworks Tool을 사용하기 위해서는 pc에 Java 프로그램이 필수 설치되어야 합니다.
Java 미설치 시 아래와 같은 알림 메세지가 나타납니다.

※ 이 파일을 열 수 없습니다.

mTworks.exe

이 파일을 열수있는 프로그램을 선택해야 합니다. 온라인에서 해당 파일을 열 수 있는 프로그램을 자동으로 검색하거나 컴퓨터에 설치되어 있는 프로그램 목록 중에서 사용자가 직접 선택할 수 있습니다.

Java 다운로드
버전 : 1.8.191 / OS : Windows

Windows JRE 다운로드
MAC용 Java 다운로드

Java 다운로드
버전 : 1.8.191 / OS : MAC

MAC JRE 다운로드

실행파일 다운로드

Java 다운로드 및 설치 > mTworks 실행파일 다운로드 > 실행 > 로그인 > 테스트 시작
※ MAC용은 압축파일 해제 후 더블클릭하시면 실행됩니다.
※ macOS Java 설치 경로 : Finder open > 이동 > 폴더로이동 > /Library/Java/JavaVirtualMachines
